# Smoke Test – Login SauceDemo

## Objetivo
Automatizar un conjunto mínimo de pruebas (Smoke Test) para validar el inicio de sesión en la aplicación SauceDemo.

## Aplicación bajo prueba
https://www.saucedemo.com/

## Tecnologías
- Cypress
- JavaScript

## Escenarios automatizados
- Login exitoso con credenciales válidas
- Login fallido con contraseña incorrecta
- Validación de campos obligatorios

## Se agrega colección de postman para el ejercicio del Modulo API
## Cómo ejecutar
1. Importar la colección en Postman
2. Importar la colection
3. Seleccionar lacolection
4. Ejecutar requests
